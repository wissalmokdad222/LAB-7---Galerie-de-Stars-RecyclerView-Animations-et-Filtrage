#  Stars Gallery - LAB 7

Ce projet est une **application Android** qui affiche une galerie de stars avec :

- RecyclerView avec animations
- Filtrage par recherche
- Notation des stars
- Affichage d’images circulaires via Glide

L’application a été développée dans le cadre du **LAB 7**.
